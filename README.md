# HHT Asset Management

Aplikasi web untuk monitoring, peminjaman, pengembalian, dan laporan penggunaan HHT.

Frontend menggunakan:

- HTML
- CSS
- JavaScript
- Bootstrap 5
- Bootstrap Icons
- SweetAlert2

Backend menggunakan:

- Google Apps Script
- Google Spreadsheet
